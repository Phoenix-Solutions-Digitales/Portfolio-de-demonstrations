# Démos — Phoenix Solutions Digitales

Sélection de démonstrations techniques et commerciales réalisées pour illustrer les prestations de **Phoenix Solutions Digitales** (Angoulême, Charente) : développement web, applications métier, marketing digital et formation.

La plupart des démos sont des fichiers autonomes (HTML / CSS / JS), sans dépendance lourde ni build, hébergeables tels quels sur GitHub Pages, Vercel ou O2switch. La démo PWA est une exception : elle nécessite plusieurs fichiers ensemble et un hébergement en HTTPS (voir sa fiche).
